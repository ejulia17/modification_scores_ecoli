# modification_effect_ecoli

This repository contains R code used to generate plots and summary analyses for modification-effect scores in *E. coli*. 

It contains plots that have been included in the the Appendix of the published version of the manuscript

"Path Signatures Enable Model-Free Mapping of RNA Modifications" by
Maud Lemercier, Paola Arrubarrena, Salvatore Di Giorgio, Julia Brettschneider, Thomas Cass, Valerie Griesche, Isabel S. Naarmann-de Vries, Anastasia Papavasiliou, Alessia Ruggieri, Irem Tellioglu, Chia Ching Wu, F. Nina Papavasiliou, Terry Lyons

The prepring is available at arXiv, section Quantitative Biology > Genomics as arXiv:2511.08855 (url: https://arxiv.org/abs/2511.08855)

The analyses in this repo compare modification-effect scores across positions relative to known RNA modification sites, including plots stratified by distance from the nearest known modification site, cluster membership, parent base, and modification type.

## Repository contents

* `R/`
  R scripts used to process the score tables and generate figures.

* `data/`
  Score/data files used as input for the plotting scripts.

* `figures/`
  Output figures generated by the R scripts, if included.

* `LICENSE.md`
  Licence information for the code and the score/data files.

## Data

The modification effect scores this repository were generated from read level scores obtained by methods introduced in the above mentioned preprint, by another contributor or contributors. The read level data files are too big to be included in a GibHub repo but can be make available to support reproducibility of the analyses and figures.

## Running the code

The scripts are written in R.

To reproduce the analyses, clone the repository, open the project directory in R or RStudio, and run the scripts in `R/`.

The scripts assume that the input score/data files are available in the `data/` directory.

## Required R packages

The code uses standard R plotting and data-handling tools. Required packages are loaded at the start of the scripts where needed.

If a package is missing, install it in R using, for example:

